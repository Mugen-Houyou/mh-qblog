📚 Elasticsearch 개요

Elasticsearch는 분산처리 검색/분석 엔진으로, 대규모 데이터의 신속한 검색 및 분석에 적합합니다. JSON 기반의 RESTful API를 사용하며, 실시간 검색, 로그 분석, 데이터 분석 등 다양한 용도로 활용됩니다.

주요 특징

  • 분산 아키텍처: 여러 노드에 걸쳐 데이터를 저장하고 검색 → 확장성과 고가용성을 제공
  • 실시간 데이터 처리: 빠른 검색 및 분석 속도로 실시간 로그 분석 등에 적합
  • 유연한 데이터 모델링: 다양한 데이터 유형과 구조를 지원하며, 복잡한 쿼리도 손쉽게 수행

설치 및 시작하기

PC에서 설치하는 것으로 가정합니다.

  1. 다운로드 및 설치: 여기에서 최신 버전을 다운로드합니다.
  2. 설치: 다운로드한 파일의 압축을 해제한 후, bin/elasticsearch 실행 파일을 실행합니다.
  3. 클러스터 구성: 기본 설정 파일(elasticsearch.yml)을 수정하여 노드를 클러스터에 연결하고, 필요한 경우 보안 설정을 적용합니다.
  4. 테스트: 브라우저나 curl 명령어를 사용하여 http://localhost:9200에 접속해 정상 작동 여부를 확인합니다.

📚 Kibana 개요

Kibana를 통해 Elasticsearch의 데이터를 시각화하고 분석할 수 있습니다. 대시보드, 차트, 지도 등을 이용하여 데이터를 직관적으로 이해할 수 있게 도와줍니다.

주요 특징

  • 대시보드 시각화: 다양한 시각화 도구를 사용하여 데이터를 효과적으로 표현
  • 실시간 분석: Elasticsearch와 연동하여 실시간 데이터를 모니터링하고 분석

설치 및 시작하기

  1. 다운로드 및 설치: Kibana 공식 사이트에서 Elasticsearch와 동일한 버전의 Kibana를 다운로드합니다.
  2. 설치: 다운로드한 파일의 압축을 해제하고, bin/kibana 실행 파일을 실행합니다.
  3. 설정: 기본 설정 파일(kibana.yml)에서 Elasticsearch 호스트 정보를 확인하고, 필요 시 추가 설정을 진행합니다.
  4. 접속: 웹 브라우저에서 http://localhost:5601에 접속하여 Kibana 대시보드를 확인합니다.

📝 마무리

Elasticsearch와 Kibana는 대규모 데이터 처리 및 시각화에 강력한 도구입니다. 이번 포스트에서는 두 도구의 기본 개념, 설치 방법, 그리고 기본적인 사용법을 살펴보았습니다. 앞으로 심화된 기능과 실제 운영 환경에서의 활용 사례를 통해 더 깊이 있는 포스트를 작성할 예정입니다.
